Quadratische Gleichungen: Zusammenfassung

Allgemeine Form und Alternative Form

Die allgemeine Form einer quadratischen Gleichung ist gegeben durch:

\[ ax^2 + bx + c = 0 \]

wobei \(a, b, c \in \mathbb{R}\) und \(a \neq 0\). Eine alternative Form der quadratischen Gleichung ist:

\[ x^2 + px + q = 0 \]

wobei \(p, q \in \mathbb{R}\).

Lösungen der quadratischen Gleichung

Die Lösungen der quadratischen Gleichung \(ax^2 + bx + c = 0\) können mit der quadratischen Formel berechnet werden:

\[ x_{1,2} = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

Die Lösungen der alternativen Form \(x^2 + px + q = 0\) sind:

\[ x_{1,2} = \frac{-p \pm \sqrt{p^2 - 4q}}{2} \]

Diskriminante

Die Diskriminante \(D\) einer quadratischen Gleichung \(ax^2 + bx + c = 0\) ist gegeben durch:

\[ D = b^2 - 4ac \]

Die Diskriminante der alternativen Form \(x^2 + px + q = 0\) ist:

\[ D = p^2 - 4q \]

Anzahl der Lösungen

Die Anzahl der Lösungen einer quadratischen Gleichung hängt von der Diskriminante ab:

  • Wenn \(D > 0\), gibt es zwei reelle, verschiedene Lösungen.
  • Wenn \(D = 0\), gibt es genau eine reelle Lösung.
  • Wenn \(D < 0\), gibt es keine reelle Lösung, sondern zwei komplexe Lösungen.

Sätze von Vieta

Für die quadratische Gleichung \(ax^2 + bx + c = 0\) mit den Lösungen \(x_1\) und \(x_2\) gelten die Sätze von Vieta:

\[ x_1 + x_2 = = -\frac{b}{a} \] \[ x_1 \cdot x_2 = \frac{c}{a} \]

Für die alternative Form \(x^2 + px + q = 0\) gelten die Sätze von Vieta:

\[ x_1 + x_2 = -p \] \[ x_1 \cdot x_2 = q \]
